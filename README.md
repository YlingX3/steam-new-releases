# Steam New Releases

Static Steam new releases report generated by `generate_report.py`.

The report keeps the newest 300 combined game and demo releases from Steam search results, enriches them with StoreBrowse details, and writes:

- `latest.html`
- `latest.json`
- `detail_cache.json`
- `run.log`

GitHub Actions refreshes the report every day at 10:00 Asia/Hong_Kong time.
